Arcline – AI Receptionist Analytics Dashboard
Internal analytics dashboard for an Australian AI receptionist company.
🔗 Live: ((https://arcline-company.vercel.app/))
Features:

Real-time call logs & duration tracking via Retell AI
Mini CRM (contact profiles, call history, notes)
AI receptionist stats (answered %, missed calls, avg handle time)
Webhook integration for post-call event updates

Stack: HTML · CSS · JavaScript · Retell AI API · Firebase (coming)
